    # 卡博替尼(cabozantinib)

    提示

    请仔细阅读说明书并在医生的指导下使用

    # 适应症

    • 适用于曾接受抗血管生成治疗的晚期肾癌
    • 适用于无法采用手术摘除的髓样甲状腺癌
    • 适用于使用多吉美耐药后的肝癌患者
    • 适用于发生骨转移的晚期前列腺癌患者
    • 肝癌、前列腺癌、卵巢癌、黑色素瘤、乳腺癌和非小细胞肺癌骨转移患者

    # 规格包装

    • 20mg*90片(¥2950)
    • 80mg*30片(¥3500)

    # 用法用量

    禁忌

    • 服用卡博替尼前至少2小时和后至少1小时不要进食。
    • 继续治疗直至疾病进展或发生不能接受毒性。
    • 整个吞服卡博替尼胶囊。不要打开卡博替尼胶囊。
    • 如出现漏服则12小时内不要再次服用。
    • 妊娠妇女卡博替尼可致胎儿危害
    • 服用卡博替尼期间不要摄取已知抑制细胞色素P450食物(如，葡萄柚，葡萄柚汁)或营养补充物。

    # 推荐剂量和服用方法

    • 甲状腺髓样癌：每次140mg，每日一次，轻度及中度肝损伤患者起始剂量为80mg
    • 肾癌、肝癌：每次60mg，每日一次
    • 骨转移：每次40mg，每日一次

    对于轻度或中度肝损伤患者，开始的剂量为80mg。 服用卡博替尼前至少2小时和后至少1小时不要进食

    # 剂量调整

    • 血栓形成事件：对心肌梗死，脑梗死，或其他严重动脉血栓栓塞事件终止卡博替尼。
    • 伤口并发症：对裂开或需要医学干预并发症不给予卡博替尼。
    • 高血压：有规律地监视血压;对高血压危象终止卡博替尼。
    • 颌骨骨坏死：终止卡博替尼。
    • 掌足红肿综合征(PPES)：中断卡博替尼，或减低卡博替尼剂量。
    • 蛋白尿：监视尿蛋白;对肾病综合征终止卡博替尼。
    • 可逆性后部白质脑病综合征(RPLS)：终止卡博替尼。

    # 不良反应

    • 腹泻(≥25%)，口腔炎，掌足红肿综合征(PPES)，体重减轻，食欲不振，恶心，疲乏，口腔痛，发色变化，味觉障碍，高血压，腹痛，和便秘。
    • 实验室异常(≥25%)是增加AST，增加 ALT，淋巴细胞减少，碱性磷酸酶增加，低钙血症，中性粒细胞减少，血小板减少, 低磷血症，和高胆红素血症。
