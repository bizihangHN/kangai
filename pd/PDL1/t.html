# T New v4.0.0+

平时日常工作中，做最多的就是写业务模块和组件。当每次新开一个view或者component的时候都需要手动创建一个新.vue，然后创建一个<template>、<script>、<style>，还是有些麻烦的。

所以在新版本中，基于plop，提供了几个基础模板，方便创建新的view或者component。

执行如下命令：

npm run new

如上面 gif 所示，现在只要轻松的点几次回车就可以轻松生成我要的基础代码片段。这里只是一个 demo，你可以按照自己需求定制模板。

有额外模板需求的可以按照plop的文档，到plop-templates目录下，创建自定义的模板。

其实这个功能和 snippets做的事情差不多。如果你觉得配置太复杂，可以安装如Vue 2 Snippets这种基于 VSCode 代码片段。
