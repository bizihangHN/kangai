    # 达伯舒(信迪利单抗注射液)

    提示

    请仔细阅读说明书并在医生的指导下使用

    # 适应症

    • 本品适用于至少经过二线系统化疗的复发或难治性经典型霍奇金淋巴瘤的治疗。

    # 规格

    • 10ml:100mg(¥2843)

    # 包装

    • 10ml/瓶(最常见)

    # 用法用量

    禁忌

    • 不得通过静脉推注或单次快速静脉注射给药
    • 肝功能不全患者禁用
    • 肾功能不全患者禁用
    • 妊娠期、哺乳期妇女禁用

    # 推荐剂量和服用方法

    采用静脉输注的方式给药，输注时间应在30~60分钟内,静脉输注的推荐剂量为200mg(体重大于60kg使用200mg,否则用100mg), 每3周给一次，直至出现疾病进展或产生不可耐受的毒性。

    抽取2瓶本品注射液(200mg), 转移到含有9mg/ml (0.9％)氯化钠溶液的静脉输液袋中，制备终浓度范围为1.5~5.0mg/ml.将稀释液轻轻翻转混匀。

    药前药品的稀释指导如下:

    • 请勿摇晃药瓶。
    • 使用前将药瓶恢复至室温(25°C或以下)。
    • 药瓶从冰箱取出后，稀释前可在室温下(25％C或以下)最长放置24小时

    # 剂量调整

    根据个体患者的安全性和耐受性，可能需要暂停给药或永久停药。不建议增加或减少剂量。

    # 不良反应

    • 发热、贫血、天门冬氨酸氨基转移酶升高、丙氨酸氨基转移酶升高、乏力、白细胞计数降低
    • 肺部感染、贫血、脂肪酶升高、血小板计数降低、肺炎、中性粒细胞计数降低、低钠血症、γ-谷氨酰转移酶升高、感染性肺炎、上消化道出血、淋巴细胞计数降低
    • 经典型霍奇金淋巴瘤
    • 免疫性肺炎、结肠炎、肝炎、肾炎、血小板减少症、心肌炎、胰腺炎、胃炎、十二指肠炎
    • 甲状腺功能减退、亢进
    • 外周神经毒性
    • 重症肌无力
