    # 可瑞达(帕博利珠单抗注射液)

    提示

    请仔细阅读说明书并在医生的指导下使用

    # 适应症

    • 博利珠单抗适用于经一线治疗失败的不可切除或转移性黑色素瘤的治疗

    # 规格包装

    将药瓶于2℃至8℃的冷藏环境下保存在原包装中，避光、避免冷冻、避免振荡

    • 100mg/4ml(¥17918)

    # 用法用量

    禁忌

    • 对本说明书(L－组氨酸，蔗糖，聚山梨酯80)的活性成份和辅料过敏者禁用
    • 帕博利珠单抗不得通过静脉推注或单次快速静脉注射给药
    • 使用前将药瓶恢复至室温（25℃或以下）

    # 推荐剂量和服用方法

    帕博利珠单抗的推荐给药方案为2mg/kg剂量静脉输注30分钟以上，每3周给药一次， 直至出现疾病进展或不可接受的毒性。

    如果患者临床症状稳定，即使有疾病进展的初步证据，但基于总体临床获益的判断，可考虑继续应用本品治疗，直至证实疾病进展。

    # 剂量调整

    不建议増加或减少剂量。对于4级或3级复发性不良反应，帕博利珠单抗应永久停药

    # 不良反应

    • 疲劳（21％），瘙痒（16％），皮疹（13％），腹泻（12％）和恶心（10％）
